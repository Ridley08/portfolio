ToDoアプリ

WEB APPLICATION / TASK MANAGER

制作概要

タスクの追加・完了チェック・削除ができるシンプルなToDoアプリです。JavaScriptのDOM操作を中心に実装し、LocalStorageを活用してページをリロードしてもデータが保持されるよう工夫しました。

こだわりポイント

タスクの追加はEnterキーでも実行できるようにし、操作性を高めました。完了済みタスクは視覚的に区別できるスタイル変化を加え、残タスク数をリアルタイムで表示する機能も実装しています。UIはミニマルにまとめ、誰でも直感的に使えることを重視しました。
