WEB SITE / LANDING PAGE

制作概要

「あなただけのAIトレーナーが理想の体を作る」をキャッチコピーに掲げた、架空のAIパーソナルトレーニングサービス「AIron」のランディングページです。体組成・睡眠・ストレスをAIが分析し、最適なトレーニングと食事プランを自動生成するサービスを想定し、ヒーロー・機能紹介・料金プラン・CTA・FAQを網羅したワンページ構成で制作しました。

こだわりポイント

ファーストビューで「AIトレーナー×自宅トレーニング」という価値提案を明確に打ち出し、スクロールに沿って機能紹介・3ステップの使い方・料金プラン・ユーザーの声・FAQと自然な流れで購入判断を後押しする構成にしました。Free・Pro・Eliteの3プラン比較表や、実績数値を並べた統計バーを設けることで信頼感を演出しています。

目的

架空サービスの申込み率（CV率）最大化。ファーストビューで離脱させず、スクロールするだけで「申し込みたい」と思わせる導線を設計することを最優先にした。

主な工夫

スマホ・PC両対応のレスポンシブ実装。3プラン比較にアンカリング（中間プランを「人気」で強調）を取り入れ、FAQで申込み前の不安を先回りして解消する構成を採用した。

得た知見

「情報の順番」がCV率に直結することを実感。機能紹介 → 実績 → 料金 → FAQ の順で並べることで、読者の「知りたい → 信頼できる → 買える → 不安ない」という心理の流れを意識的に設計できた。
